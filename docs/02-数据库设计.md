# MES 数据库设计

## 1. 设计原则

- 所有表以 `mes_` 前缀命名
- 主键使用雪花ID（Snowflake）或 UUID v7
- 审计字段：`created_at`, `created_by`, `updated_at`, `updated_by`
- 逻辑删除：`is_deleted` 布尔字段
- 关键业务表使用事务保证一致性

## 2. 核心表结构

### 2.1 工厂与产线

```sql
-- 工厂
CREATE TABLE mes_factory (
    id          BIGINT PRIMARY KEY,
    code        VARCHAR(50) NOT NULL UNIQUE,     -- 工厂编码
    name        VARCHAR(200) NOT NULL,            -- 工厂名称
    address     VARCHAR(500),
    status      TINYINT DEFAULT 1,               -- 1:启用 0:停用
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
    is_deleted  TINYINT DEFAULT 0
);

-- 车间
CREATE TABLE mes_workshop (
    id          BIGINT PRIMARY KEY,
    factory_id  BIGINT NOT NULL,
    code        VARCHAR(50) NOT NULL,
    name        VARCHAR(200) NOT NULL,
    status      TINYINT DEFAULT 1,
    FOREIGN KEY (factory_id) REFERENCES mes_factory(id)
);

-- 产线
CREATE TABLE mes_production_line (
    id          BIGINT PRIMARY KEY,
    workshop_id BIGINT NOT NULL,
    code        VARCHAR(50) NOT NULL,
    name        VARCHAR(200) NOT NULL,
    line_type   VARCHAR(50),                     -- 流水线/单元线/固定工位
    status      TINYINT DEFAULT 1,
    FOREIGN KEY (workshop_id) REFERENCES mes_workshop(id)
);

-- 工位
CREATE TABLE mes_workstation (
    id          BIGINT PRIMARY KEY,
    line_id     BIGINT NOT NULL,
    code        VARCHAR(50) NOT NULL,
    name        VARCHAR(200) NOT NULL,
    seq_no      INT DEFAULT 0,                   -- 工位序号（流水线用）
    status      TINYINT DEFAULT 1,
    FOREIGN KEY (line_id) REFERENCES mes_production_line(id)
);
```

### 2.2 物料与 BOM

```sql
-- 物料主数据
CREATE TABLE mes_material (
    id          BIGINT PRIMARY KEY,
    code        VARCHAR(50) NOT NULL UNIQUE,       -- 物料编码 (MaxLength=50)
    name        VARCHAR(300) NOT NULL,              -- 物料名称
    spec        VARCHAR(200),                       -- 规格型号 (MaxLength=200)
    unit        VARCHAR(20),                        -- 单位（个/套/KG等），可空
    category    VARCHAR(50),                        -- 物料分类 (MaxLength=50)
    bom_level   INT DEFAULT 0,                      -- BOM层级
    stock_qty   DECIMAL(18,2) DEFAULT 0,            -- 库存数量
    status      TINYINT DEFAULT 1,
    created_at  TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    updated_at  TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    is_deleted  TINYINT DEFAULT 0
);

-- BOM 物料清单
CREATE TABLE mes_bom (
    id            BIGINT PRIMARY KEY,
    product_id    BIGINT NOT NULL,               -- 成品/半成品物料ID
    material_id   BIGINT NOT NULL,               -- 子件物料ID
    quantity      DECIMAL(18,6) NOT NULL,         -- 用量
    scrap_rate    DECIMAL(5,2) DEFAULT 0,         -- 损耗率(%)
    seq_no        INT DEFAULT 0,                  -- 工序序号
    valid_from    DATE,
    valid_to      DATE,
    status        TINYINT DEFAULT 1,
    FOREIGN KEY (product_id) REFERENCES mes_material(id),
    FOREIGN KEY (material_id) REFERENCES mes_material(id)
);
```

### 2.3 工艺路线

```sql
-- 工艺路线头
CREATE TABLE mes_routing (
    id            BIGINT PRIMARY KEY,
    material_id   BIGINT NOT NULL,               -- 对应物料
    routing_code  VARCHAR(50) NOT NULL,
    routing_name  VARCHAR(200),
    version       VARCHAR(20) DEFAULT 'V1.0',    -- 版本号 (如 V1.0)
    status        TINYINT DEFAULT 1,
    FOREIGN KEY (material_id) REFERENCES mes_material(id)
);

-- 工艺路线明细（工序）
CREATE TABLE mes_routing_step (
    id            BIGINT PRIMARY KEY,
    routing_id    BIGINT NOT NULL,
    step_no       INT NOT NULL,                   -- 工序号
    step_name     VARCHAR(200) NOT NULL,           -- 工序名称
    workstation_type VARCHAR(100),                -- 要求工位类型
    standard_time DECIMAL(10,2),                  -- 标准工时(分钟)
    is_qc_point   TINYINT DEFAULT 0,              -- 是否质检点
    is_scrap_point TINYINT DEFAULT 0,              -- 是否废品判定点
    FOREIGN KEY (routing_id) REFERENCES mes_routing(id)
);
```

### 2.4 工单与报工（核心业务）

```sql
-- 生产工单
CREATE TABLE mes_work_order (
    id              BIGINT PRIMARY KEY,
    order_no        VARCHAR(50) NOT NULL UNIQUE,   -- 工单号
    source_type     VARCHAR(20),                    -- 来源: ERP/MANUAL
    source_ref      VARCHAR(100),                   -- 来源单号（ERP订单号）
    material_id     BIGINT NOT NULL,                -- 生产物料
    routing_id      BIGINT,                         -- 工艺路线
    planned_qty     DECIMAL(18,2) NOT NULL,         -- 计划数量
    completed_qty   DECIMAL(18,2) DEFAULT 0,        -- 已完成数量
    scrap_qty       DECIMAL(18,2) DEFAULT 0,        -- 报废数量
    status          INT NOT NULL DEFAULT 0,         -- 工单状态 (枚举整型存储)
    -- status 枚举: PENDING(0) / RELEASED(1) / SCHEDULED(2) / IN_PROGRESS(3) / ON_HOLD(4) / COMPLETED(5) / CLOSED(6) / CANCELLED(7)
    plan_start_time TIMESTAMPTZ,                    -- 计划开始时间
    plan_end_time   TIMESTAMPTZ,                    -- 计划结束时间
    actual_start_time TIMESTAMPTZ,
    actual_end_time TIMESTAMPTZ,
    priority        INT DEFAULT 50,                 -- 优先级: LOW(0) / NORMAL(50) / HIGH(80) / URGENT(100)
    factory_id      BIGINT,
    workshop_id     BIGINT,
    line_id         BIGINT,
    assignee        VARCHAR(100),                   -- 负责人
    remark          TEXT,
    rework_from_id  BIGINT,                         -- 返工来源工单ID
    created_at      TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    updated_at      TIMESTAMPTZ DEFAULT CURRENT_TIMESTAMP,
    is_deleted      TINYINT DEFAULT 0
);

-- 工单工序（工单拆分后的工序级任务）
CREATE TABLE mes_work_order_step (
    id              BIGINT PRIMARY KEY,
    work_order_id   BIGINT NOT NULL,
    step_no         INT NOT NULL,
    step_name       VARCHAR(200) NOT NULL,
    workstation_id  BIGINT,
    planned_qty     DECIMAL(18,2) NOT NULL,
    completed_qty   DECIMAL(18,2) DEFAULT 0,
    scrap_qty       DECIMAL(18,2) DEFAULT 0,
    status          VARCHAR(30) DEFAULT 'PENDING',
    plan_start_time DATETIME,
    plan_end_time   DATETIME,
    FOREIGN KEY (work_order_id) REFERENCES mes_work_order(id)
);

-- 报工记录
CREATE TABLE mes_work_report (
    id              BIGINT PRIMARY KEY,
    report_no       VARCHAR(50) NOT NULL UNIQUE,   -- 报工单号
    work_order_id   BIGINT NOT NULL,
    step_id         BIGINT,                         -- 关联工序
    workstation_id  BIGINT,                         -- 报工工位
    operator_id     BIGINT,                         -- 操作工 (外键关联 mes_user.id)
    report_type     VARCHAR(20),                    -- COMPLETE / REWORK / SCRAP
    good_qty        DECIMAL(18,2) DEFAULT 0,        -- 合格数量
    scrap_qty       DECIMAL(18,2) DEFAULT 0,        -- 报废数量
    rework_qty      DECIMAL(18,2) DEFAULT 0,        -- 返工数量
    duration_min    INT,                            -- 工时(分钟)
    batch_no        VARCHAR(100),                   -- 批次号
    report_time     TIMESTAMPTZ NOT NULL,
    remark          TEXT,
    FOREIGN KEY (work_order_id) REFERENCES mes_work_order(id),
    FOREIGN KEY (operator_id) REFERENCES mes_user(id)
);
```

### 2.5 质量检测

```sql
-- 质检单
CREATE TABLE mes_qc_inspection (
    id              BIGINT PRIMARY KEY,
    inspect_no      VARCHAR(50) NOT NULL UNIQUE,
    source_type     VARCHAR(30),                    -- INCOMING / FIRST / PATROL / FINAL
    source_ref      VARCHAR(100),                   -- 来源（工单号/采购单号）
    work_order_id   BIGINT,
    material_id     BIGINT,
    inspector       VARCHAR(100),
    inspect_result  VARCHAR(20),                    -- PASS / FAIL / PENDING
    inspect_time    DATETIME,
    created_at      DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 质检项
CREATE TABLE mes_qc_inspection_item (
    id              BIGINT PRIMARY KEY,
    inspection_id   BIGINT NOT NULL,
    item_name       VARCHAR(200) NOT NULL,
    spec_value      VARCHAR(200),                   -- 规格值
    actual_value    VARCHAR(200),                   -- 实测值
    result          VARCHAR(20),                    -- PASS / FAIL
    FOREIGN KEY (inspection_id) REFERENCES mes_qc_inspection(id)
);
```

## 3. 枚举值定义

### 3.1 工单状态 (WorkOrderStatus)

| 值 | 名称 | 说明 |
|---|------|------|
| 0 | PENDING | 待下达 |
| 1 | RELEASED | 已下达 |
| 2 | SCHEDULED | 已排产 |
| 3 | IN_PROGRESS | 生产中 |
| 4 | ON_HOLD | 暂停 |
| 5 | COMPLETED | 已完成 |
| 6 | CLOSED | 已关闭 |
| 7 | CANCELLED | 已取消 |

### 3.2 优先级 (Priority)

| 值 | 名称 | 说明 |
|---|------|------|
| 0 | LOW | 低优先级 |
| 50 | NORMAL | 普通 |
| 80 | HIGH | 高优先级 |
| 100 | URGENT | 紧急 |

### 3.3 质检结果 (InspectionResult)

| 值 | 说明 |
|---|------|
| ACCEPT | 接受 |
| REWORK | 返工 |
| SCRAP | 报废 |
| CONCESSION | 特采 |

## 4. 索引策略

- 工单号、物料编码、批次号、序列号：唯一索引
- 外键关联字段：普通索引
- 时间范围查询字段：复合索引
- 状态字段：单列索引

## 4. 数据归档策略

- 超过 6 个月的已关闭工单自动归档到 `mes_work_order_archive`
- 报工记录按季度分表（`mes_work_report_YYYY_QQ`）
- 追溯记录永久保留
