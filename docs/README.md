# MES Framework 设计文档

按类别组织的设计文档，每类文件夹内有 `README.md` 说明该类文档概览。

| 目录 | 类别 | 说明 |
|------|------|------|
| [01-项目概述](01-项目概述/) | 业务层 | 需求分析、项目背景、AI 融合方案 |
| [02-技术设计](02-技术设计/) | 技术层 | 架构、数据库、接口、DDD 重构方案 |
| [03-项目管理](03-项目管理/) | 管理层 | 开发计划、修复方案与计划 |
| [04-质量保障](04-质量保障/) | 质量层 | 测试计划、CI/CD 方案 |

### 文档创建规则

- **修复类文档**：执行修复任务时，在 `03-项目管理/修复/` 下创建
  - 方案文件：`YYYY-MM-DD-XXX修复方案.md`
  - 计划文件：`YYYY-MM-DD-XXX修复执行计划.md`
- **技术方案**：放在 `02-技术设计/` 下
- **执行计划**：放在 `03-项目管理/` 下
