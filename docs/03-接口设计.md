# MES 接口设计（RESTful API）

## 1. 接口规范

- 基础路径：`/api/v1`
- 请求/响应：JSON
- 认证：JWT Bearer Token
- 分页格式：`{ page, page_size, total, total_pages, items[] }`
- 统一响应：`{ code, message, data, trace_id }`

## 3. 接口示例

### 创建工单

```
POST /api/v1/work-orders

Request:
{
    "material_id": 100001,
    "routing_id": 200001,
    "planned_qty": 500,
    "plan_start_time": "2026-05-10T08:00:00",
    "plan_end_time": "2026-05-12T18:00:00",
    "priority": 50,
    "source_type": "ERP",
    "source_ref": "PO-2026-05001",
    "remark": "加急订单"
}

Response:
{
    "code": 0,
    "message": "success",
    "data": {
        "id": 300001,
        "order_no": "MO-20260508-0001",
        "status": "PENDING",
        "created_at": "2026-05-08T12:00:00"
    }
}
```

### 报工

```
POST /api/v1/work-reports

    "work_order_id": 300001,
```
